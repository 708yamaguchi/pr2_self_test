Alpha 2.0 Caster

Verifying Correct Caster Type

This will only work with Alpha 2.0 casters. Make sure your caster looks like the one in the picture above. If it has plate sheet metal on the sides around the wheels, it is the correct caster. To qualify an Alpha 1.0 caster, go back and select 'Caster Alpha 1.0' from the menu after you scan the part.

Do not proceed if you are not sure.

If your caster spins extremely quickly during calibration and hysteresis, it is probably an Alpha 1.0 caster. You must fail the component if this happens, and retry the test as an Alpha 1.0 caster.

Checking Caster Front/Back

The front of the caster is the side with the continuous sheet metal front (the side with the barcode in the picture). From this, you should be able to identify the left and right wheels. Make sure the correct wheel is turning during left/right hysteresis tests.

Connecting the Caster

Connect the caster to the test cart's power cable using the attached female power cable. Connect the EtherCAT to the test cart connection using the M-to-M RJ-45 adapter.

Press 'Continue' to start qualification.

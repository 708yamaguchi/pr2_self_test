Experimental Test

This is an experimental test. It has not been proven, use with caution.

Running Test

Stand clear for calibration. This test will move the arm to a series of positions, and measure torques for the shoulder lift and elbow flex joints.

Connecting Arm

Connect the full arm by plugging in the shoulder pan MCB using the EtherCAT and power from the test cart.
